# Gerador de JSON para uso do AA ONE web
# 📖 Como Usar
Abra o ficheiro index.html em qualquer navegador moderno.

Preencha os campos com as informações da loja/terminal.

Clique em Gerar Arquivo JSON.

Copie o código gerado ou faça o download do ficheiro.

Siga as instruções na secção "Como aplicar o JSON gerado" dentro da própria ferramenta para inserir os dados no LocalStorage do site de autoatendimento.

# 💻 Instalação e Hospedagem
Como o projeto consiste em ficheiros estáticos, não requer servidor para correr:

Local: Basta clonar a pasta e abrir o index.html.

Hospedagem: publica na Vercel, pode ser acessado através do link: https://gerador-json-linx.vercel.app/.

# 🔒 Segurança e Privacidade
Os dados inseridos nesta ferramenta não são enviados para nenhum servidor externo. Todo o processamento e armazenamento do histórico ocorre localmente no navegador do utilizador.
